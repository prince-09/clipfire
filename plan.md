# AI Video Repurposer — MVP Product Plan

## Product Vision

A creator tool that converts long videos (podcasts, interviews, lectures) into viral short-form clips. Upload 1 video, get 10-30 ready-to-post shorts.

## Target Users

- Podcast creators
- YouTube educators
- Interview channels
- Coaches / influencers
- Content repurposing agencies

---

## MVP Features

### 1. Video Upload
- Drag & drop upload (MP4, MOV, MKV — up to 2GB)
- YouTube URL paste (download via yt-dlp)
- Store locally for dev, S3 for prod

### 2. AI Clip Detection
- Extract audio via FFmpeg
- Transcribe with Whisper (word-level timestamps)
- Send transcript to LLM (via OpenRouter) to find viral moments
- Classify each clip: Strong Hook, Educational Insight, Debate/Opinion, Funny Moment, Storytelling, Emotional Peak, Quotable Moment
- Score each clip 0-10 (viral potential)
- Return 10-30 clips per video

### 3. Clip Review & Edit
- List view with cards showing: title, type badge, viral score, transcript excerpt
- Click to preview clip in video player
- Trim start/end
- Select/deselect clips for export

### 4. Hook + Caption Generator
- Per clip: 3 hook options, 1 title, 1 caption, 5-10 hashtags
- Generated by LLM alongside clip detection

### 5. Styled Captions
- Auto-generated from word-level Whisper timestamps
- 3 styles: classic subtitle, bold-pop, minimal
- Position: top / center / bottom
- Burned into video via FFmpeg on export

### 6. Export
- Vertical crop (9:16) via FFmpeg center-crop
- Square (1:1) option
- Captions burned in or exported as SRT
- Single clip download or batch ZIP

---

## Processing Pipeline

```
1. Upload video
2. FFmpeg → extract audio (WAV)
3. Whisper → transcript + word timestamps
4. LLM (OpenRouter/Mistral) → detect clips + score + hooks/captions
5. Save clips to MongoDB
6. User reviews clips in UI
7. User selects clips → FFmpeg renders vertical + captions
8. Download as MP4 or ZIP
```

## Data Models

```
Project
├── title, status, sourceUrl, videoPath
├── durationSeconds, errorMessage
├── createdAt
├── Transcript (fullText, wordTimestamps[], language)
└── Clips[]

Clip
├── title, startTime, endTime, durationSeconds
├── segmentType, confidenceScore
├── viralScore, scoreBreakdown (JSON)
├── hooks (JSON), caption, hashtags (JSON)
├── isSelected, exportStatus, outputPath
└── Exports[]

Export
├── format (9:16 | 1:1 | 16:9)
├── captionStyle, filePath, fileSizeBytes
└── createdAt
```

## NOT in MVP

- Auth / login (skipped for now)
- BYOK (platform uses its own key)
- Auto-posting to social media
- Face tracking / auto-reframe (center crop only)
- Split screen layouts
- B-roll generation
- Analytics dashboards
- Team collaboration
- Batch processing

---

## Success Criteria

1. Upload a 30-60 min video
2. Get 10-30 clips automatically in < 5 minutes
3. Review clips with viral scores
4. Export vertical 9:16 shorts with captions
5. Total workflow under 10 minutes
